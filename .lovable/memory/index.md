# Project Memory

## Core
- German-language site for organic farmer (Meister der Landwirtschaft). Emphasize regionality & credibility.
- Distribution: local collection (Abholung) and delivery (Lieferung).
- Stack: React, Tailwind CSS, Headless Shopify (Storefront API), Zustand (cart persistence).
- Checkout flow redirects to Shopify.
- Contact form emails use Resend.
- Rule: Display unit pricing (e.g., '€/kg') with German labels on all product cards/pages.
